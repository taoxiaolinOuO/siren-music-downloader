# siren-music-downloader

作为方舟7年的老玩家，塞壬唱片一直是我的最爱，质量很高，所以写了一个程序，用来爬取塞壬唱片官网的音乐数据到本地，可以自己搭建一个塞壬唱片音乐库。

# 功能

- 双击Releases里的 Exe程序，默认选中全量下载
- 点击开始按钮，就会爬取塞壬唱片官网的音乐，就会在 `当前Exe程序` 所在目录下，生成一个siren-music文件夹，保存音乐数据
- 每次运行Exe程序，都会检查当前目录是有有已下载的音乐文件，如果有，会自动跳过，不会重复下载
- 下载音乐文件到本地，对音乐做了分类处理，按照专辑分类命名
- 还有最近更新的功能，每次运行Exe程序之后，会自动检查是否有新的音乐数据，如果有，会自动下载到最近更新的文件夹里
